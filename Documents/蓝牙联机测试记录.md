# 蓝牙联机测试记录

## 一、测试目标
验证锄大地蓝牙联机流程是否可用于课程演示：
1. 主机创建房间。
2. 客户端通过已配对设备加入房间。
3. 主机分配座位，客户端准备。
4. 主机开局并广播种子、公共快照和私人手牌。
5. 客户端出牌/过牌请求由主机校验后同步。
6. 断开连接时能显示离线状态。
7. 心跳超时和重连入口可用于异常恢复演示。

## 三、测试前准备
1. 所有设备安装同一个 `app-debug.apk`。
2. 所有设备打开系统蓝牙。
3. 客户端手机先在系统蓝牙设置中与主机手机完成配对。
4. 打开 App 后点击“授权蓝牙”，确认权限已允许。

## 五、已知限制
1. 当前只列出系统已配对设备，还没有做附近设备扫描。
2. 心跳会携带玩家座位号；主机可将超时座位标记离线，但仍需要真机长时间测试确认稳定性。
3. 重连目前是手动入口，可恢复座位和对局状态；后台自动重连尚未实现。
4. 当前 UI 以调试演示为主，附近设备扫描还没有实现。
